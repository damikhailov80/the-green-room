# Project Rules

## Language Standards

1. **Code, comments, and UI must be in English**
   - All code: English only
   - All code comments: English only
   - All user-facing UI text (buttons, labels, messages): English only
   - Chat responses to user: in the language of user's request

2. **No markdown files without explicit request**
   - Do not create .md files automatically
   - Only create markdown files when user explicitly asks for them
